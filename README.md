# 🌐 Portfolio Web Developer

Website portfolio personal dengan desain modern, elegan, dan responsif. Dibuat untuk menampilkan profil, skill, dan project dengan tampilan clean + aesthetic.

---

## 🚀 Fitur Utama

- 🎯 Hero section dengan animasi smooth
- 🧑‍💻 About section (profil + skill)
- 📂 Portfolio project grid
- 🔗 Social media integration
- ✨ Animasi scroll (Intersection Observer)
- 📱 Fully responsive (mobile friendly)
- 🎨 Custom styling (dark theme + elegant UI)

---

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3 (Custom styling, Flexbox, Grid)
- JavaScript (Vanilla JS)
- Google Fonts (Montserrat & Cormorant Garamond)

---

## 📁 Struktur Halaman

- **Home** → Landing page / hero section  
- **About** → Deskripsi diri + skill  
- **Projects** → Showcase project  
- **Contact** → Social media & kontak
